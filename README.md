# idk_example

1. Ensure idk_build, idk_core, idk_gfx and idk_platform are placed adjacent to this project.
2. Build the project:
    ```bash
    # Usage
    idk_build/build.sh --target idk_example <--debug|--release> <--clean>

    # Example
    idk_build/build.sh --target idk_example --release --clean
    ```
